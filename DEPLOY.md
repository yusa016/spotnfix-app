# Deploy SpotN'Fix

**Quick start:** see [GO_LIVE.md](./GO_LIVE.md)

**GitHub:** https://github.com/yusa016/spotnfix-app

**Stack:** Render (Node web service) + TiDB Cloud Serverless (MySQL-compatible database)

Netlify and Firebase are **not** suitable for this project (needs Express + MariaDB/MySQL).

Railway works but free credits expire; Render free tier keeps your URL permanently (app may sleep when idle).
